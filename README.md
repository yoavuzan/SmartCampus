# 🚀 SmartCampus - Full Stack Management System

SmartCampus is a powerful, modern web application designed for campus management. It leverages a high-performance Python backend and a lightning-fast React frontend to provide a seamless user experience.

## 🛠️ Tech Stack

### Backend
- **FastAPI**: High-performance Python web framework.
- **SQLModel**: Modern SQL databases interaction (Pydantic + SQLAlchemy).
- **Uvicorn**: ASGI server for production-ready performance.
- **SQLite**: Lightweight relational database.

### Frontend
- **React (Vite)**: Fast and modern frontend development environment.
- **Tailwind CSS**: Utility-first CSS framework for rapid UI styling.
- **Axios**: For seamless API communication.

## 🌟 Key Features
- **Fast API endpoints** with automatic Swagger documentation.
- **Responsive Design** optimized for all devices.
- **Type Safety** from the database to the frontend.
- **Real-time Updates** via optimized state management.
